### 把图片从页面上全部提取出来!

而且把一些图片还原成原图，譬如 微博 和 淘宝 ！  
尤其是 花瓣，可以在列表页直接获取原图！  
<s>(希望花瓣别屏蔽吧哈哈哈哈哈哈哈哈哈哈</s>

使用方法：  
任意页面点击书签后，出现图片列表页面；
点击 esc 键，可移除添加的图片列表页，  
不过可能会对原页面造成稍微的破坏，刷新即可。
